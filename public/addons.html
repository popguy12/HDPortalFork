About the addons

Most if not all people tend to play Hideous Destructor with at least one addon. Usually a sprite/sound replacement or a weapon. This page contains mostly HD addons, and a few non-HD addons that work well with the mod.

Some addons may be outdated. Hideous Destructor is updated constantly, so compatibility is not always guaranteed. If the mod does not work with a stable release, try the master branch.

If a file is a zip, chances are the real .pk3 is inside, unless downloaded from GitHub/GitLab/etc. Keep that in mind if loading the zip doesn't do anything.

Dates are MM/DD/YY.
