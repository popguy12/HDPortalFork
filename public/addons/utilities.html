Utilities

4.26mm Bulk Ammo Box

by valerie_v
Requires Ugly as Sin

This 4.26mm bulk box is great for storing stray 4.26mm to be used in reloading your 7.76mm rounds, or for keeping a ready cache for inserting into PD-42 magazines and other gray-market weapons capable of using the UAC/Volt 4.26mm Caseless Round.

Bulk: Full - 128, Empty - 32. Capacity: 2000 rounds. "Compression" in container results in 75% reduction in bulk.

Will always spawn empty in backpack and ammo box spawn pools (1/5 chance). Cannot be given through loadout because it will always, in such case, be given full.

Grappling Hook

by Josh771 (edited by Accensus)

Edited version of Josh's grappling hook. The only difference in this version is that it no longer uses the Use key as a toggle, and instead offers a custom keybind. The jump key also no longer acts as a secondary detachment key. Instead, it's a modifier, so Jump + Toggle = "detatch and gain minor vertical velocity".

Ace's note: please, PLEASE do not ping me about this thing. It was a one-off edit that I got asked to do and I do not want to touch this ever again.

UAC Corporate Armour

by HalfBakedCake

Corporate armour, UAC body armour given to corporate security to help defend against angry investors.

- Corporate armour can only be found in backpacks.

- Loadout code is awc for worn armour, arc for spare.

- This armour is more protective than combat armour, as light as garrison armour, and has environmental resistance on par with the radsuit.

- The armour self-heals damage to it over time. It only self-heals when worn.

Preview.
