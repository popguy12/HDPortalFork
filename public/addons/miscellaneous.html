AI Director

by Josh771
Does not require HD

A different approach to dynamic monster / item spawning, as compared to Demon Counterstrike. Attempts intelligent guesses about the player's state and actions when spawning enemies. Uses existing monster spawnpoints as reference nodes (instead of the player's past locations), and uses the level's existing monster composition for spawnlists (so if a map has lots of imps, then the director will use lots of imps). Simpler configuration options make it easier to tweak for desired difficulty and flow.
